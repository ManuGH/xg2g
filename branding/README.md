# xg2g Branding Assets

This directory contains official branding assets for the xg2g project.

## Directory Structure

```
branding/
├── hero/           # Large format logos with tagline
├── flat/           # Simple flat versions
├── icon/           # Icon-only versions
└── README.md       # This file
```

## Usage Guidelines

### 1. Hero Logo (`hero/xg2g-hero-dark.svg`)

**Use for:**
- GitHub README header
- GitHub Social Preview (OpenGraph)
- Marketing materials
- Project presentations

**Specifications:**
- Size: 1200×400px
- Background: Dark (#0f172a)
- Includes tagline: "GATEWAY-TO-GATEWAY STREAMING PROXY"
- Metallic gradient effect

**Example (Markdown):**
```markdown
<div align="center">
  <img src="branding/hero/xg2g-hero-dark.svg" alt="xg2g Logo" width="900"/>
</div>
```

### 2. Flat Logos (`flat/`)

**xg2g-flat-white.svg** - White-on-dark version
- Use for: Dark backgrounds, inline documentation
- Size: 200×200px
- No tagline, no text

**xg2g-flat-black.svg** - Black-on-light version
- Use for: Light backgrounds, print materials
- Size: 200×200px
- No tagline, no text

### 3. Icon (`icon/xg2g-icon.svg`)

**Use for:**
- Favicons
- App icons
- Small UI elements
- Profile pictures

**Specifications:**
- Size: 200×200px
- Symbol only (no text)
- Gradient background

## Design Principles

### Colors

**Primary Blue:**
- Light: `#60a5fa`
- Medium: `#3b82f6`
- Dark: `#1e40af`

**Background:**
- Dark: `#0f172a` (slate-900)
- Medium: `#1e293b` (slate-800)

**Text:**
- White: `#ffffff`
- Gray: `#94a3b8` (slate-400)

### Symbol Meaning

- **Play Button**: Streaming functionality
- **Signal Waves**: Broadcast/streaming signals
- **Circular Form**: Gateway concept, continuity

## Brand Positioning

xg2g is positioned as an **infrastructure-grade streaming gateway**, emphasizing:
- Stability
- Engineering quality
- Professional operation
- Production readiness

## File Format Notes

All assets are provided in SVG format for:
- Scalability without quality loss
- Version control compatibility
- Small file sizes
- Web compatibility
